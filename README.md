# ShawarmaLegend

## 專案簡介

ShawarmaLegend 是一個用於檢測和計算沙威瑪數量的自動化工具。當你在遊玩《沙威瑪傳奇》時，這個程式可以幫助你自動捕捉遊戲畫面並分析沙威瑪的數量。

## 功能

- 自動捕捉指定遊戲視窗的畫面
- 檢測畫面中的沙威瑪數量
- 實時顯示沙威瑪的數量變化

## 安裝

請確保你的環境中已安裝 Python 3.x，然後使用以下命令安裝所需的依賴包：

```bash
pip install -r requirements.txt
```

## 使用方法

1. 確保遊戲正在運行並且視窗標題正確。
2. 執行 `沙威瑪檢測器.py` 檔案。
3. 程式將自動捕捉畫面並計算沙威瑪的數量。

## 注意事項

- 請確保你的螢幕解析度和遊戲視窗大小與程式設計時的設定相符，以獲得最佳效果。
- 若無法找到指定的視窗，請檢查視窗標題是否正確。
